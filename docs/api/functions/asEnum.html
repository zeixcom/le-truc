Function: asEnum()#

asEnum(valid): Parser<string, UI>

Defined in: src/parsers/string.ts:29

Parser that constrains an attribute value to one of a fixed set of allowed strings.

Comparison is case-insensitive. If the attribute value is absent or does not match any allowed value, the first entry of valid is returned as the default.

Parameters#

valid#

[string, ...string[]]

Non-empty array of allowed values; first entry is the default

Returns#

Parser<string, UI>

Parser that returns a valid enum value
