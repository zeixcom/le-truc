Function: bindClass()#

bindClass<T>(element, token, transform?): (value) => void

Defined in: src/helpers.ts:63

Returns a function that toggles a CSS class token on an element.

value=true adds the token; value=false removes it. If transform is provided, it converts the incoming value to a boolean first.
