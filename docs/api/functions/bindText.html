Function: bindText()#

bindText(element, preserveComments?): (value) => void

Defined in: src/bindings.ts:129

Returns a function that sets the text content of an element.

When preserveComments is true, uses setTextPreservingComments to retain HTML comment nodes. When false (default), sets el.textContent directly. Numbers are coerced to strings via String().
