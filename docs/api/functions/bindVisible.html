Function: bindVisible()#

bindVisible<T>(element, transform?): (value) => void

Defined in: src/helpers.ts:88

Returns a function that controls element visibility via el.hidden = !value.

value=true makes the element visible; value=false hides it. Matches the direction of the v1.0 show() effect. If transform is provided, it converts the incoming value to a boolean first.
