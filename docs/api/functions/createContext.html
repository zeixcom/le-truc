Function: createContext()#

createContext<V>(key): Context<string, V>

Defined in: src/helpers/context.ts:133

Create a typed context key.

The Context type brands the key type with the __context__ property that carries the type of the value the context references. This helper function creates a properly typed context key from a plain value.

V

Parameters#
