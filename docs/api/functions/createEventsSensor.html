Function: createEventsSensor()#

createEventsSensor<T, E>(target, init, events): Sensor<T>

Defined in: src/events.ts:176

Create a Sensor<T> driven by DOM events on a target element.

Use this inside expose() as a property initializer when a single reactive value should be derived from events on a specific element. The listener is attached directly to target; the handler receives { event, target, prev }.
