createSlot<T>(initialSignal, options?): Slot<T>

Defined in: node_modules/@zeix/cause-effect/types/src/nodes/slot.d.ts:44

Creates a slot signal that delegates its value to a swappable backing signal.

A slot acts as a stable reactive source that can be used as a property descriptor via Object.defineProperty(target, key, slot). Subscribers link to the slot itself, so replacing the backing signal with replace() invalidates them without breaking existing edges. Setter calls forward to the current backing signal when it is writable.
