Function: defineComponent()#

defineComponent<P, U>(name, props?, select?, setup?): Component<P>

Defined in: src/component.ts:80

Define and register a reactive custom element.

Calls customElements.define() and returns the registered class. Reactive properties are initialised in connectedCallback and torn down in disconnectedCallback.
