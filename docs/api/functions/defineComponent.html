Function: defineComponent()#

defineComponent<P>(name, factory): HTMLElement & P

Defined in: src/component.ts:129

Define and register a reactive custom element using the v1.1 factory form.

The factory receives a FactoryContext at connect time: query helpers (first, all), the host element, and expose() for declaring reactive properties. It returns a flat array of effect descriptors created by helpers like watch(), on(), pass(), provideContexts(), and requestContext().

Effects activate after dependency resolution — child custom elements are guaranteed to be defined before any descriptor runs.
