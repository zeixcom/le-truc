Function: pass()#

pass<P, Q>(props): Effect<P, Component<Q>>

Defined in: src/effects/pass.ts:42

Effect for passing reactive values to a descendant Le Truc component by replacing the backing signal of the target's Slot.

No cleanup/restore is needed: when the parent unmounts, the child is torn down as well. For re-parenting scenarios, use context instead.
