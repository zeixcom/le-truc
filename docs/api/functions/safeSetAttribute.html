safeSetAttribute(element, attr, value): void

Set an attribute on an element with security validation.

Blocks on* event handler attributes and validates URL-like values against a safe-protocol allowlist (http:, https:, ftp:, mailto:, tel:). Violations throw a descriptive error — they are never silent.
