Function: schedule()#

schedule(element, task): void

Defined in: src/scheduler.ts:41

Schedule a task to be executed on the next animation frame, with automatic deduplication per component. If the same component schedules multiple tasks before the next frame, only the latest task will be executed.

Element

Element for deduplication
