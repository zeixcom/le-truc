Function: setStyle()#

setStyle<P, E>(prop, reactive?): Effect<P, E>

Defined in: src/effects/style.ts:17

Effect for setting a CSS custom property or inline style on an element.

When the reactive value is null, the style property is removed via el.style.removeProperty(prop). Otherwise it is set via el.style.setProperty(prop, value).

Parameters#

Returns#
