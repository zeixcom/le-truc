Function: unown()#

unown<T>(fn): T

Defined in: node_modules/@zeix/cause-effect/types/src/graph.d.ts:230

Runs a callback without any active owner. Any scopes or effects created inside the callback will not be registered as children of the current active owner (e.g. a re-runnable effect). Use this when a component or resource manages its own lifecycle independently of the reactive graph.
