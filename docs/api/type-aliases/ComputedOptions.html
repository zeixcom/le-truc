watched()?#

optional watched: (invalidate) => Cleanup

Optional callback invoked when the signal is first watched by an effect. Receives an invalidate function that marks the signal dirty and triggers re-evaluation. Must return a cleanup function that is called when the signal is no longer watched.

This enables lazy resource activation for computed signals that need to react to external events (e.g. DOM mutations, timers) in addition to tracked signal dependencies.
