FactoryResult = (EffectDescriptor | FactoryResult | Falsy)[]

Defined in: src/effects.ts:46

The return value of the factory function.

An array of effect descriptors (and optional falsy guards for conditional effects). Nested arrays are automatically flattened. Falsy values (false, undefined, null, "", 0) are filtered out before activation, enabling the element && [watch(...)] conditional pattern.
