The props argument of defineComponent — a map from property names to their initializers.

Each value may be:

  • A static value or Signal — used directly as the initial signal value.
  • A Parser (branded with asParser()) — called with the attribute value string at connect time; for 4-param form also on every attribute change.
  • A MethodProducer (branded with defineMethod()) — assigned directly as the property value; the function IS the method. Per-instance state lives in factory scope.
