Type Alias: SensorEventHandler<T, Evt, E>#

SensorEventHandler<T, Evt, E> = (event, element, prev) => MaybePromise<T | void>

Defined in: src/events.ts:35

Handler for a single event type inside createEventsSensor.

Receives:

  • event — the original DOM event (typed to the specific event type)
  • element — the matched element (properly typed, unlike event.target)
  • prev — the current sensor value before this event

Return the new sensor value to update it, or void / Promise<void | T> to leave the value unchanged. Async handlers are awaited; the resolved value updates the sensor if non-void.
