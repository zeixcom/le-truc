The watch helper type in FactoryContext.

Drives a reactive effect from a signal source (property name, Signal, thunk, or array). Only the declared sources trigger re-runs — incidental reads inside the handler are not tracked. Returns an EffectDescriptor.

Thunk form () => T is wrapped in createComputed, so all signals read inside it are tracked in the pure phase — useful for deriving or transforming values before the side-effectful handler runs.
