createCollection function

Function: createCollection()#

createCollection<T>(watched, options?): Collection<T>

Defined in: node_modules/@zeix/cause-effect/types/src/nodes/collection.d.ts:52

Creates an externally-driven Collection with a watched lifecycle. Items are managed via the applyChanges(changes) helper passed to the watched callback. The collection activates when first accessed by an effect and deactivates when no longer watched.
